This page lists the services available to the ASF. (Services only used by the Infrastructure team, such as internal SQL databases and FreeBSD tinderboxes, are not listed.) DNS is the only authoritative record of which service is on which machine, so please use the service names ("home.apache.org") instead of the machine names ("minotaur.apache.org") when using the services.

As this page will periodically grow stale, a dynamic page of the underlying hosts & services can prove useful. Additionally there is information about the machines themselves.

Web sites¶

The site-dev mailing list manages the following websites:

  • ASF websites

    • projects.apache.org - an index into all the Projects at the ASF

    • www.apache.org - the main ASF website (build instructions)

    • home.apache.org - details about the individual committers at the ASF

  • Committer personal web space : http://home.apache.org/~login/

Email¶

  • Mail server - QMail/QSMTPD

  • mailing lists - EZMLM

  • public mailing list archives (mod_mbox)

  • private mailing list archives (mod_mbox) and search. (ASF Members have full access; PMC members have access to their PMC's archives only, via the latter link.) (draft search syntax documentation)

  • Spam control - Spamfilter + SpamAssassin

  • Mail forwarding: See id.apache.org

Issue tracking¶

There are currently two issue tracking systems in use at the ASF. Additionally, security vulnerabilities are mailed to security@apache.org.

  • Bugzilla (manual)

  • Jira (manual, ASF FAQ, Svn/Git integration)

  • Bug reporting guidelines

For historical reasons, some projects have their own instances of Jira or Bugzilla. See issues.apache.org for a list.

Source repository¶

  • Subversion (SVN) repositories

  • ViewVC (Browser interface to the main repository)

  • Guide for commiters and other community members

  • Read-only Git mirrors of SVN codebases

  • Experimental writable Git repositories

Repository to Issuetracker Integrations¶

  • Subversion to Bugzilla Robot
  • Svn/Git to JIRA

Source repository publisher/subscriber services¶

  • SvnPubSub
  • GitPubSub

Shell (SSH) Server¶

people.apache.org has been retired. There is no replacement shell access planned for committers.

Note that the existing shell accounts will remain available on minotaur.apache.org for a short while.

Virtual servers¶

We offer FreeBSD jails and Ubuntu VMs (no separate link yet).

A few PMCs still have Solaris zones, but those are deprecated (i.e. we don't create any more of those).

Distributions¶

  • current distributions

  • historical distribution archives

  • rsync (for distribution mirrors)

  • Mirror Monitor - public mirrors status page

  • Nexus

  • Archiva

  • Pear
